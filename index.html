키워드가 거래내역 내용에 포함되면 해당 카테고리로 자동 분류됩니다. 변경 후 전체 재분류 적용을 누르면 기존 내역에도 적용됩니다.

🚫 제외 키워드
이 키워드가 포함된 거래는 업로드 시 제외로 표시됩니다 (가져오기 시 자동 제외).
💡 단, 키워드 자동분류 규칙이 있으면 분류 규칙이 우선 적용됩니다.

카카오뱅크 카카오뱅크 모바일 ▼
🔑 이메일 수신 파일은 암호화 방식 문제로 앱에서 직접 열 수 없습니다.
파일을 이 Claude 채팅에 첨부하면 복호화 파일을 만들어 드립니다. 복호화된 파일을 앱에 업로드하세요.
  1. 카카오뱅크 앱 실행 → 홈에서 계좌 탭
  2. 계좌 상세 화면 우측 상단 ⚙️ 설정 아이콘 탭
  3. 계좌관리 화면에서 거래내역 다운로드 탭
  4. 이메일 확인 후 거래내역 기간 설정 (최대 1년)
  5. 개인정보 수집 및 이용에 동의합니다 체크
  6. 받기 버튼 탭 → 이메일로 수신
  7. 이메일 첨부파일을 저장 후 앱에 업로드
토스뱅크 토스뱅크 모바일 ▼
🔑 이메일 수신 파일은 암호화 방식 문제로 앱에서 직접 열 수 없습니다.
파일을 이 Claude 채팅에 첨부하면 복호화 파일을 만들어 드립니다. 복호화된 파일을 앱에 업로드하세요.
  1. 토스 앱 실행 → 홈에서 토스뱅크 계좌 탭
  2. 계좌 상세 화면 상단 관리 탭
  3. 목록에서 증명서 발급 탭
  4. 토스뱅크 증명서 발급하기 목록에서 거래내역서 탭
  5. 거래내역서 안내 화면에서 다음 버튼 탭
  6. 기간 설정 후 이메일 입력 → 발급 요청
  7. 이메일 첨부파일을 저장 후 앱에 업로드

카카오페이 카카오페이 모바일 ▼
🔑 이메일 수신 파일은 암호화되어 있어 Claude 채팅에 첨부 → 복호화 요청 후 앱에 업로드해야 합니다.
  1. 카카오페이 앱 실행 → 홈 우측 상단 ≡ 메뉴 탭
  2. 메뉴 화면에서 고객센터 탭
  3. 고객센터 화면에서 증명서 발급 탭
  4. 페이아이 화면에서 거래내역서 발급 탭
  5. 카카오페이머니 거래내역서 화면에서
    • 시작일 / 종료일 설정
    • 파일 전송 형태 → 엑셀 선택
    • 마스킹 여부 → 해제 선택
    • 이메일 주소 입력
  6. 발송요청 버튼 탭 → 이메일 수신 대기
  7. 수신한 파일을 이 Claude 채팅에 첨부하여 복호화 요청
  8. Claude가 제공한 카카오페이_복호화.xlsx를 앱에 업로드
